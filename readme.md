# Learn AI from Claude

A hands-on 6-week AI engineering curriculum built with FastAPI, Ollama, vector databases, and RAG.

## Structure
- `week1/` — FastAPI + Ollama + streaming API
- `week2/` — Embeddings + Qdrant vector database + semantic search
- `week3/` — Full RAG application + React chat UI
- `week4/` — Agents + tool calling + conversation memory
- `week5/` — Deploy to OCI/OKE + CI/CD pipeline
- `week6/` — Evaluation, re-ranking, and polish

## Stack
Python · FastAPI · Ollama · Qdrant · LlamaIndex · ReactJS · OCI
